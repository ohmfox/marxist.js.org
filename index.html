The Classless Javascript movement

What is this?

ES6 added the class keyword to Javascript. We think that this adds far too much complexity to Javascript and that the built in "class-like" abilities of Javascript are much more clear and powerful.

Let's see an example

Someone recently pointed me to this code: var aClass = arg => class { constructor() { console.log(arg); } }; new (aClass("ey"))(); //prints "ey" and retuns an object At first glance this looks pretty weird, since there is a lot of obfuscation happening here. However, this code translates to something very simple to read and understand. var aClass = (arg) => { return function() { console.log(arg); } }
